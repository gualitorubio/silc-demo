# ⚖️ SILC - Sistema de Inteligencia Legal y Contexto
**Versión 17.4.12.25**

SILC es una plataforma avanzada de **Legal-Tech** desarrollada por **Rubio Intelligence Systems**. Utiliza una arquitectura de Generación Aumentada por Recuperación (RAG) para ofrecer consultas jurídicas de alta precisión, conectando la legislación federal vigente con un profundo acervo histórico mexicano.

## 🚀 Capacidades del Sistema
- **Análisis Evolutivo:** Contrasta normativas actuales con leyes de los siglos XIX y XX.
- **Base de Datos Vectorial:** Acceso a más de 91,000 registros indexados en Pinecone.
- **Razonamiento IA:** Impulsado por Google Gemini 1.5 Flash para interpretaciones estratégicas.

## 🛠️ Stack Tecnológico
- **Lenguaje:** Python 3.10+
- **Interfaz:** Streamlit Cloud
- **Modelos de Lenguaje:** Gemini API
- **Embeddings:** Multilingual-E5-Large
- **Base de Datos:** Pinecone (Vector Search)

## 📁 Estructura del Repositorio
- `app.py`: Lógica de la interfaz de usuario y conexión con LLM.
- `requirements.txt`: Librerías necesarias para el despliegue.
- `.gitignore`: Filtro de archivos temporales de Python.

## 🛡️ Seguridad y Privacidad
Este repositorio es **Privado**. Las llaves de acceso (API Keys) están protegidas mediante variables de entorno (Secrets) en el servidor de despliegue, garantizando que los datos sensibles nunca estén expuestos en el código fuente.
